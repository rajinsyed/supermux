бһҒбҹ’бһүбһ»бҹҶбһ”бһ¶бһ“бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ orchestrator бһҹбһҡбһҹбҹҒбһҡбһҖбһјбһҠбһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһ—бһ¶бһӮбһ…бҹ’бһҡбһҫбһ“бһ“бҹғбһ–бһҪбһҖбһңбһ¶бһҮбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Electron/Tauri бһ бһҫбһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҡбһ”бһҹбҹӢбһңбһ¶бһ”бһ¶бһ“бһ’бҹ’бһңбһҫбһІбҹ’бһҷбһҒбҹ’бһүбһ»бҹҶбһ’бһ»бһүбҹ” бһҒбҹ’бһүбһ»бҹҶбһҖбҹҸбһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮбһ…бһјбһӣбһ…бһ·бһҸбҹ’бһҸ terminal бһ…бҹ’бһҡбһҫбһ“бһҮбһ¶бһ„ бһҠбҹ„бһҷбһҹбһ¶бһҡ orchestrator бһ”бҹ’бһҡбһ—бҹҒбһ‘ GUI бһ…бһ¶бһҖбҹӢбһҹбҹ„бһўбҹ’бһ“бһҖбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ workflow бһҡбһ”бһҹбҹӢбһ–бһҪбһҖбһңбһ¶бҹ” бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһҒбҹ’бһүбһ»бҹҶбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸ cmux бһҮбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё macOS бһҠбһҫбһҳбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Swift/AppKitбҹ” бһңбһ¶бһ”бҹ’бһҡбһҫ libghostty бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһү terminal бһ бһҫбһҷбһўбһ¶бһ“ config Ghostty бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ theme бһ–бһ»бһҳбҹ’бһ–бһўбһҖбҹ’бһҹбһҡ бһ“бһ·бһ„бһ–бһҺбҹҢбҹ”

бһҖбһ¶бһҡбһ”бһ“бҹ’бһҗбҹӮбһҳбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бһӮбһә sidebar бһ“бһ·бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„бҹ” sidebar бһҳбһ¶бһ“бһ•бҹ’бһ‘бһ¶бҹҶбһ„бһ”бһүбҹ’бһҲбһҡбһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһү git branch бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–/бһӣбҹҒбһҒ PR бһҠбҹӮбһӣбһ—бҹ’бһҮбһ¶бһ”бҹӢ бһҗбһҸбһ’бҹ’бһңбһҫбһҖбһ¶бһҡ port бһҠбҹӮбһӣбһҖбҹҶбһ–бһ»бһ„бһҹбҹ’бһҠбһ¶бһ”бҹӢ бһ“бһ·бһ„бһўбһҸбҹ’бһҗбһ”бһ‘бһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„бһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷбһ”бҹҶбһ•бһ»бһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ workspace бһ“бһёбһҳбһҪбһҷбҹ—бҹ” бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„бһ…бһ¶бһ”бҹӢбһҷбһҖбһӣбҹҶбһҠбһ¶бһ”бҹӢ terminal (OSC 9/99/777) бһ бһҫбһҷбһҳбһ¶бһ“ CLI (`cmux notify`) бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ—бҹ’бһҮбһ¶бһ”бҹӢбһ…бһјбһӣбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„ hook бһҡбһ”бһҹбҹӢбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Claude Code, OpenCode бҹ”бһӣбҹ” бһ“бҹ…бһ–бҹҒбһӣбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҖбҹҶбһ–бһ»бһ„бһҡбһ„бҹӢбһ…бһ¶бҹҶ бһ•бҹ’бһ‘бһ¶бҹҶбһ„бһҡбһ”бһҹбҹӢбһңбһ¶бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһҡбһ„бҹ’бһңбһ„бҹӢбһ–бһҺбҹҢбһҒбҹҖбһң бһ бһҫбһҷбһ•бҹ’бһ‘бһ¶бҹҶбһ„бһ—бҹ’бһӣбһәбһЎбһҫбһ„бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ sidebar бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһҒбҹ’бһүбһ»бҹҶбһўбһ¶бһ…бһ”бҹ’бһҡбһ¶бһ”бҹӢбһҗбһ¶бһҳбһҪбһҷбһҺбһ¶бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҒбҹ’бһүбһ»бҹҶ бһҖбһ¶бһҸбҹӢбһҶбҹ’бһӣбһ„бһ•бҹ’бһ‘бһ¶бҹҶбһ„бһ”бҹҶбһ”бҹӮбһҖ бһ“бһ·бһ„бһ•бҹ’бһ‘бһ¶бҹҶбһ„бһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ” Cmd+Shift+U бһӣбҹ„бһҸбһ‘бҹ…бһҹбһ¶бһҡбһҠбҹӮбһӣбһҳбһ·бһ“бһ‘бһ¶бһ“бҹӢбһўбһ¶бһ“бһҗбҹ’бһҳбһёбһ”бҹҶбһ•бһ»бһҸбҹ”

browser бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҳбһ¶бһ“ API бһҠбҹӮбһӣбһўбһ¶бһ…бһҹбһҡбһҹбҹҒбһҡ script бһ”бһ¶бһ“бһҠбҹӮбһӣбһ•бҹ’бһ‘бҹҒбһҡбһҳбһҖбһ–бһё [agent-browser](https://github.com/vercel-labs/agent-browser)бҹ” бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһўбһ¶бһ…бһҗбһҸ snapshot бһ“бҹғ accessibility tree бһ‘бһ‘бһҪбһӣ ref бһ“бҹғ element бһ…бһ»бһ… бһ”бҹҶбһ–бҹҒбһү form бһ“бһ·бһ„бһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғ JSбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹҶбһ”бҹӮбһҖбһ•бҹ’бһ‘бһ¶бҹҶбһ„ browser бһ“бҹ…бһҮбһ¶бһ”бҹӢбһ“бһ№бһ„ terminal бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ бһҫбһҷбһІбҹ’бһҷ Claude Code бһ’бҹ’бһңбһҫбһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳбһҮбһ¶бһҳбһҪбһҷ dev server бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢбҹ”

бһўбҹ’бһңбһёбҹ—бһӮбҹ’бһҡбһ”бҹӢбһҷбҹүбһ¶бһ„бһўбһ¶бһ…бһҹбһҡбһҹбҹҒбһҡ script бһ”бһ¶бһ“бһҸбһ¶бһҳбһҡбһҷбҹҲ CLI бһ“бһ·бһ„ socket API вҖ” бһ”бһ„бҹ’бһҖбһҫбһҸ workspace/бһ•бҹ’бһ‘бһ¶бҹҶбһ„ бһ”бҹҶбһ”бҹӮбһҖбһ•бҹ’бһ‘бһ¶бҹҶбһ„ бһ•бҹ’бһүбһҫ keystroke бһ”бһҫбһҖ URL бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ browserбҹ”

## бһ бҹ’бһҹбһ·бһ“ (Zen) бһ“бҹғ cmux

cmux бһҳбһ·бһ“бһҖбҹҶбһҺбһҸбҹӢбһ…бҹ’бһ”бһ¶бһ”бҹӢбһўбҹҶбһ–бһёбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҖбһ¶бһ“бҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһҡбһ”бһҹбҹӢбһ–бһҪбһҖбһӮбҹҒбһ‘бҹҒбҹ” бһңбһ¶бһҮбһ¶ terminal бһ“бһ·бһ„ browser бһҮбһ¶бһҳбһҪбһҷ CLI бһ бһҫбһҷбһ•бҹ’бһ“бҹӮбһҖбһҠбҹӮбһӣбһ“бҹ…бһҹбһӣбҹӢбһӮбһәбһўбһ¶бһҹбҹ’бһҡбҹҗбһҷбһӣбһҫбһўбҹ’бһ“бһҖбҹ”

cmux бһӮбһәбһҮбһ¶ primitive бһҳбһ·бһ“бһҳбҹӮбһ“бһҮбһ¶бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ‘бҹҒбҹ” бһңбһ¶бһ•бҹ’бһҠбһӣбҹӢбһІбҹ’бһҷбһўбҹ’бһ“бһҖбһ“бһјбһң terminal бһҳбһҪбһҷ browser бһҳбһҪбһҷ бһҖбһ¶бһҡбһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„ workspace бһҖбһ¶бһҡбһ”бҹҶбһ”бҹӮбһҖ бһ•бҹ’бһ‘бһ¶бҹҶбһ„ бһ“бһ·бһ„ CLI бһҳбһҪбһҷбһҠбһҫбһҳбҹ’бһ”бһёбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһңбһ¶бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбҹ” cmux бһҳбһ·бһ“бһ”бһ„бҹ’бһҒбҹҶбһўбҹ’бһ“бһҖбһ…бһјбһӣбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһңбһ·бһ’бһёбһҠбҹӮбһӣбһҳбһ¶бһ“бһӮбҹҶбһ“бһ·бһҸбһ•бҹ’бһҠбһ¶бһ…бҹӢбһҖбһ¶бһҡбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҹбһҡбһҹбҹҒбһҡбһҖбһјбһҠбһ‘бҹҒбҹ” бһўбҹ’бһңбһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҹбһ¶бһ„бһҹбһ„бҹӢбһҮбһ¶бһҳбһҪбһҷ primitive бһӮбһәбһҮбһ¶бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

бһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһҸбҹӮбһ„бһҸбҹӮбһ”бһ¶бһ“бһҹбһ¶бһ„бһҹбһ„бҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“бһҡбһ”бһҹбҹӢбһ–бһҪбһҖбһӮбҹҒбҹ” бһӮбҹ’бһҳбһ¶бһ“бһ“бһҡбһҺбһ¶бһҳбҹ’бһ“бһ¶бһҖбҹӢбһ”бһ¶бһ“бһҡбһҖбһғбһҫбһүбһңбһ·бһ’бһёбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ“бҹ…бһЎбһҫбһҷбһ‘бҹҒ бһ бһҫбһҷбһҖбҹ’бһҡбһ»бһҳбһҠбҹӮбһӣбһҖбҹҶбһ–бһ»бһ„бһҹбһ¶бһ„бһҹбһ„бҹӢбһ•бһӣбһ·бһҸбһ•бһӣбһ”бһ·бһ‘бһҮбһ·бһҸбһҖбҹҸбһ”бҹ’бһҡбһ¶бһҖбһҠбһҮбһ¶бһҳбһ·бһ“бһ”бһ¶бһ“бһҡбһҖбһғбһҫбһүбһҠбҹӮбһҡбҹ” бһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҠбҹӮбһӣбһҮбһ·бһҸбһҹбҹ’бһ“бһ·бһ‘бҹ’бһ’бһ”бҹҶбһ•бһ»бһҸбһ“бһ№бһ„ codebase бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“бһҡбһ”бһҹбҹӢбһ–бһҪбһҖбһӮбҹҒбһ“бһ№бһ„бһҡбһҖбһғбһҫбһүбһңбһ¶бһҳбһ»бһ“бһӮбҹҒбҹ”

бһ•бҹ’бһҠбһӣбҹӢбһІбҹ’бһҷбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҳбһҪбһҷбһӣбһ¶бһ“бһ“бһ¶бһҖбҹӢбһ“бһјбһң primitive бһҠбҹӮбһӣбһўбһ¶бһ…бһ•бҹ’бһӮбһ»бҹҶбһ”бһ¶бһ“ бһ бһҫбһҷбһ–бһҪбһҖбһӮбҹҒбһ“бһ№бһ„бһҡбһҪбһҳбһӮбҹ’бһ“бһ¶бһҡбһҖбһғбһҫбһү workflow бһҠбҹӮбһӣбһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһ”бҹҶбһ•бһ»бһҸбһӣбһҝбһ“бһҮбһ¶бһ„бһҖбҹ’бһҡбһ»бһҳбһ•бһӣбһ·бһҸбһ•бһӣбһҺбһ¶бһҳбһҪбһҷбһўбһ¶бһ…бһҡбһ…бһ“бһ¶бһ–бһёбһӣбһҫбһ…бһ»бҹҮбһҖбҹ’бһҡбҹ„бһҳбҹ”

## бһҜбһҖбһҹбһ¶бһҡ

бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ”бһ“бҹ’бһҗбҹӮбһҳбһўбҹҶбһ–бһёбһҡбһ”бҹҖбһ”бһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ cmux [бһҹбһјбһҳбһҳбһҖбһҖбһ¶бһ“бҹӢбһҜбһҖбһҹбһ¶бһҡбһҡбһ”бһҹбҹӢбһҷбһҫбһ„](https://cmux.com/docs/getting-started?utm_source=readme)бҹ”

бһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“вҖӢбһҳбһ¶бһ“вҖӢбһҸбҹӮ macOS бһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбҹ” cmux бһӮбһәвҖӢбһҮбһ¶вҖӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё native Swift + AppKitбҹ”

### бһҸбһҫвҖӢбһҳбһ¶бһ“вҖӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё iOS бһҠбҹӮбһҡвҖӢбһ¬вҖӢбһ‘бҹҒ?

бһҳбһ¶бһ“ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮвҖӢбһ“бҹ…вҖӢбһҖбҹ’бһ“бһ»бһ„вҖӢбһҠбҹҶбһҺбһ¶бһҖбҹӢвҖӢбһҖбһ¶бһӣ betaбҹ” бһ•бҹ’бһӮбһјбһ•бҹ’бһӮбһ„ iPhone бһҡбһ”бһҹбҹӢвҖӢбһўбҹ’бһ“бһҖвҖӢбһҮбһ¶бһҳбһҪбһҷ Mac бһҡбһ”бһҹбҹӢвҖӢбһўбҹ’бһ“бһҖвҖӢбһ–бһёвҖӢбһ”бһ„бҹ’бһўбһҪбһ… Mobile Connect бһҡбһҪбһ…вҖӢбһ—бҹ’бһҮбһ¶бһ”бҹӢвҖӢбһ‘бҹ… terminals бһҡбһ”бһҹбҹӢвҖӢбһўбҹ’бһ“бһҖвҖӢбһ–бһёвҖӢбһ‘бһјбһҡбһҹбҹҗбһ–бҹ’бһ‘вҖӢбһҡбһ”бһҹбҹӢвҖӢбһўбҹ’бһ“бһҖ бһҠбҹ„бһҷвҖӢбһҳбһ¶бһ“вҖӢбһҮбһҳбҹ’бһҡбһҫбһҹвҖӢбһ”бһүбҹ’бһҮбһјбһ“вҖӢбһ”бһ“бҹ’бһҸвҖӢбһҖбһ¶бһҡвҖӢбһҮбһјбһ“вҖӢбһҠбҹҶбһҺбһ№бһ„вҖӢбһҡбһ”бһҹбҹӢ terminalбҹ” бһңбһ¶вҖӢбһҸбҹ’бһҡбһјбһңвҖӢбһ”бһ¶бһ“вҖӢбһ•бҹ’бһҠбһӣбҹӢвҖӢбһҮбһјбһ“вҖӢбһ“бҹ…вҖӢбһӣбһҫ TestFlight бһҮбһ¶ cmux BETAбҹ” бһҹбһјбһҳвҖӢбһҳбһҫбһӣ [бһҜбһҖбһҹбһ¶бһҡ iOS](https://cmux.com/docs/ios)бҹ”

### бһҸбһҫ cmux бһҠбҹҶбһҺбһҫбһҡвҖӢбһҖбһ¶бһҡвҖӢбһҮбһ¶бһҳбһҪбһҷ coding agents бһўбҹ’бһңбһёбһҒбҹ’бһӣбҹҮ?

бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбҹ” cmux бһӮбһәвҖӢбһҮбһ¶ terminal бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮ agent бһҺбһ¶бһҳбһҪбһҷвҖӢбһҠбҹӮбһӣвҖӢбһҠбҹҶбһҺбһҫбһҡвҖӢбһҖбһ¶бһҡвҖӢбһ“бҹ…вҖӢбһҖбҹ’бһ“бһ»бһ„ terminal бһҠбҹҶбһҺбһҫбһҡвҖӢбһҖбһ¶бһҡвҖӢбһ”бһ¶бһ“вҖӢбһ—бҹ’бһӣбһ¶бһҳбҹ—бҹ– Claude Code, Codex, OpenCode, Gemini CLI, Kiro, Aider, Goose, Amp, Cline, Cursor Agent бһ“бһ·бһ„вҖӢбһўбҹ’бһңбһёвҖӢбһ•бҹ’бһҹбҹҒбһ„вҖӢбһ‘бҹҖбһҸвҖӢбһҠбҹӮбһӣвҖӢбһўбҹ’бһ“бһҖвҖӢбһўбһ¶бһ…вҖӢбһ…бһ¶бһ”бҹӢвҖӢбһ•бҹ’бһҠбһҫбһҳвҖӢбһ–бһё command lineбҹ”

### бһҸбһҫ cmux бһўбһ¶бһ…вҖӢбһҡбҹҖбһ”бһ…бҹҶ agents бһ“бһ·бһ„ subagents бһ…бҹ’бһҡбһҫбһ“вҖӢбһ”бһ¶бһ“вҖӢбһ¬вҖӢбһ‘бҹҒ?

бһ”бһ¶бһ“бҹ” бһ“бҹ…вҖӢбһ–бҹҒбһӣвҖӢбһҠбҹӮбһӣ agent бһҳбһҪбһҷвҖӢбһ”бһ„бҹ’бһҖбһҫбһҸ subagents бһ¬вҖӢбһҹбһҳбһ¶бһҮбһ·бһҖвҖӢбһҖбҹ’бһҡбһ»бһҳ cmux бһ”бҹҶбһ”бҹ’бһӣбҹӮбһ„вҖӢбһ–бһҪбһҖвҖӢбһңбһ¶вҖӢбһ‘бҹ…вҖӢбһҮбһ¶ native panes бһ“бһ·бһ„ splits бһҮбҹҶбһ“бһҪбһҹвҖӢбһұбҹ’бһҷвҖӢбһҠбҹҶбһҺбһҫбһҡвҖӢбһҖбһ¶бһҡвҖӢбһ•бҹ’бһ‘бҹғвҖӢбһҒбһ¶бһ„вҖӢбһҖбҹ’бһҡбҹ„бһҷвҖӢбһҠбҹӮбһӣвҖӢбһӣбһ¶бһҖбҹӢвҖӢбһ”бһ¶бҹҶбһ„бҹ” бһңбһ¶вҖӢбһӮбһ¶бҹҶбһ‘бҹ’бһҡ [Claude Code teams](https://cmux.com/docs/agent-integrations/claude-code-teams) бһ“бһ·бһ„вҖӢбһҖбһ¶бһҡвҖӢбһҡбҹҖбһ”бһ…бҹҶвҖӢ multi-model бһҡбһ”бһҹбҹӢ [oh-my-opencode](https://cmux.com/docs/agent-integrations/oh-my-opencode) бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮ agent бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢвҖӢбһ“бҹ…вҖӢбһҖбҹ’бһ“бһ»бһ„вҖӢбһҖбһ¶бһҡвҖӢбһҠбҹҶбһҺбһҫбһҡвҖӢбһҖбһ¶бһҡвҖӢбһҳбһҪбһҷвҖӢбһўбһ¶бһ…вҖӢбһҳбһҫбһӣвҖӢбһғбһҫбһүвҖӢбһ“бһ·бһ„вҖӢбһӮбҹ’бһҡбһ”бҹӢвҖӢбһӮбҹ’бһҡбһ„вҖӢбһ”бһ¶бһ“бҹ”

### бһҸбһҫвҖӢбһҒбҹ’бһүбһ»бҹҶвҖӢбһўбһ¶бһ…вҖӢбһ”бҹ’бһҡбһҫ cmux бһҮбһ¶бһҳбһҪбһҷвҖӢбһҳбҹүбһ¶бһҹбҹҠбһёбһ“вҖӢбһ–бһёвҖӢбһ…бһҳбҹ’бһ„бһ¶бһҷвҖӢбһ”бһ¶бһ“вҖӢбһ¬вҖӢбһ‘бҹҒ?

бһ”бһ¶бһ“бҹ” бһ”бһҫбһҖ workspaces бһҸбһ¶бһҳвҖӢбһҡбһҷбҹҲ SSH бһ бһҫбһҷвҖӢбһ—бҹ’бһҮбһ¶бһ”бҹӢвҖӢбһ‘бҹ…вҖӢбһҹбһҳбҹҗбһҷ tmux бһ–бһёвҖӢбһ…бһҳбҹ’бһ„бһ¶бһҷ бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮ agents бһўбһ¶бһ…вҖӢбһҠбҹҶбһҺбһҫбһҡвҖӢбһҖбһ¶бһҡвҖӢбһ“бҹ…вҖӢбһӣбһҫвҖӢбһҳбҹүбһ¶бһҹбҹҠбһёбһ“вҖӢбһ–бһёвҖӢбһ…бһҳбҹ’бһ„бһ¶бһҷвҖӢбһҒбһҺбҹҲвҖӢбһҠбҹӮбһӣвҖӢбһўбҹ’бһ“бһҖвҖӢбһ”бһүбҹ’бһҮбһ¶вҖӢбһ–бһҪбһҖвҖӢбһңбһ¶вҖӢбһ–бһё cmuxбҹ” бһҹбһјбһҳвҖӢбһҳбһҫбһӣ [SSH бһ“бһ·бһ„вҖӢбһ–бһёвҖӢбһ…бһҳбҹ’бһ„бһ¶бһҷ](https://cmux.com/docs/ssh)бҹ”

### бһҸбһҫвҖӢбһҖбһ¶бһҡвҖӢбһҮбһјбһ“вҖӢбһҠбҹҶбһҺбһ№бһ„вҖӢбһҠбҹҶбһҺбһҫбһҡвҖӢбһҖбһ¶бһҡвҖӢбһҷбҹүбһ¶бһ„вҖӢбһҠбһјбһ…бһҳбҹ’бһҠбҹҒбһ…?

бһ“бҹ…вҖӢбһ–бҹҒбһӣвҖӢбһҠбҹӮбһӣвҖӢбһҠбҹҶбһҺбһҫбһҡвҖӢбһҖбһ¶бһҡвҖӢбһҳбһҪбһҷвҖӢбһҸбҹ’бһҡбһјбһңвҖӢбһҖбһ¶бһҡвҖӢбһҖбһ¶бһҡвҖӢбһҷбһҖвҖӢбһ…бһ·бһҸбҹ’бһҸвҖӢбһ‘бһ»бһҖвҖӢбһҠбһ¶бһҖбҹӢ cmux бһ”бһ„бҹ’бһ бһ¶бһүвҖӢбһҡбһ„бҹ’бһңбһ„бҹӢвҖӢбһҮбһјбһ“вҖӢбһҠбҹҶбһҺбһ№бһ„вҖӢбһҮбһ»бҹҶвҖӢбһңбһ·бһү panes бһҹбҹ’бһӣбһ¶бһҖвҖӢбһҳбһ·бһ“вҖӢбһ‘бһ¶бһ“бҹӢвҖӢбһўбһ¶бһ“вҖӢбһ“бҹ…вҖӢбһҖбҹ’бһ“бһ»бһ„ sidebar popover бһҮбһјбһ“вҖӢбһҠбҹҶбһҺбһ№бһ„ бһ“бһ·бһ„вҖӢбһҖбһ¶бһҡвҖӢбһҮбһјбһ“вҖӢбһҠбҹҶбһҺбһ№бһ„ desktop macOSбҹ” бһҖбһ¶бһҡвҖӢбһ‘бһ¶бҹҶбһ„вҖӢбһ“бҹҒбҹҮвҖӢбһ”бһүбҹ’бһ…бҹҒбһүвҖӢбһҠбҹ„бһҷвҖӢбһҹбҹ’бһңбҹҗбһҷвҖӢбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·вҖӢбһҸбһ¶бһҳвҖӢбһҡбһҷбҹҲ escape sequences бһҹбҹ’бһҠбһ„бҹӢбһҠбһ¶бһҡвҖӢбһҡбһ”бһҹбҹӢ terminal (OSC 9/99/777) бһ¬вҖӢбһўбҹ’бһ“бһҖвҖӢбһўбһ¶бһ…вҖӢбһ’бҹ’бһңбһҫвҖӢбһұбҹ’бһҷвҖӢбһ–бһҪбһҖвҖӢбһңбһ¶вҖӢбһҠбҹҶбһҺбһҫбһҡвҖӢбһҖбһ¶бһҡвҖӢбһҮбһ¶бһҳбһҪбһҷ [cmux CLI](https://cmux.com/docs/notifications#cli-usage) бһ“бһ·бһ„ [agent hooks](https://cmux.com/docs/notifications#integration-examples)бҹ” agent бһҺбһ¶бһҳбһҪбһҷвҖӢбһҠбҹӮбһӣвҖӢбһӮбһ¶бҹҶбһ‘бҹ’бһҡ hooks бһ¬ OSC бһҠбҹҶбһҺбһҫбһҡвҖӢбһҖбһ¶бһҡвҖӢбһ”бһ¶бһ“ бһҡбһҪбһҳвҖӢбһ‘бһ¶бҹҶбһ„ Claude Code, Codex, OpenCode бһ“бһ·бһ„ piбҹ”

### бһҸбһҫ cmux бһўбһ¶бһ…вҖӢбһҹбһҡбһҹбҹҒбһҡвҖӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёвҖӢбһ”бһ¶бһ“вҖӢбһ¬вҖӢбһ‘бҹҒ?

бһ”бһ¶бһ“бҹ” бһҡбһ¶бһӣбҹӢвҖӢбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–вҖӢбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢвҖӢбһҳбһ¶бһ“вҖӢбһҸбһ¶бһҳвҖӢбһҡбһҷбҹҲ cmux CLI бһ“бһ·бһ„ Unix socket бһҳбһҪбһҷбҹ– бһ”бһ„бҹ’бһҖбһҫбһҸ workspaces бһ”бһҫбһҖ split panes бһ•бҹ’бһүбһҫ input бһўбһ¶бһ“вҖӢбһҳбһ¶бһҸбһ·бһҖбһ¶вҖӢбһўбҹҒбһҖбҹ’бһҡбһ„бҹӢ бһҗбһҸвҖӢбһҡбһјбһ”вҖӢбһўбҹҒбһҖбҹ’бһҡбһ„бҹӢ бһ“бһ·бһ„вҖӢбһ”бһүбҹ’бһҮбһ¶вҖӢ browser бһҖбҹ’бһ“бһ»бһ„вҖӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбҹ” бһҹбһјбһҳвҖӢбһҳбһҫбһӣ [бһҜбһҖбһҹбһ¶бһҡвҖӢбһҷбҹ„бһ„ CLI](https://cmux.com/docs/api) бһ“бһ·бһ„вҖӢбһҜбһҖбһҹбһ¶бһҡ [browser automation](https://cmux.com/docs/browser-automation)бҹ”

### бһҸбһҫ browser бһҠбҹӮбһӣвҖӢбһҳбһ¶бһ“вҖӢбһҹбҹ’бһҡбһ¶бһ”бҹӢвҖӢбһўбһ¶бһ…вҖӢбһ’бҹ’бһңбһҫвҖӢбһўбҹ’бһңбһёвҖӢбһ”бһ¶бһ“вҖӢбһҒбҹ’бһӣбҹҮ?

cmux бһўбһ¶бһ…вҖӢ бһ”бҹҶбһ”бҹӮбһҖвҖӢ browser pane бһ–бһ·бһҸвҖӢбһ”бҹ’бһҡбһ¶бһҖбһҠвҖӢбһҳбһҪбһҷвҖӢбһ“бҹ…вҖӢбһҮбһ¶бһ”бҹӢвҖӢбһ“бһ№бһ„ terminal бһҡбһ”бһҹбҹӢвҖӢбһўбҹ’бһ“бһҖ бһ бһҫбһҷвҖӢбһңбһ¶вҖӢбһўбһ¶бһ…вҖӢбһҹбһҡбһҹбҹҒбһҡвҖӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёвҖӢбһ”бһ¶бһ“вҖӢбһ–бҹҒбһүбһӣбҹҒбһүбҹ– бһҡбһ»бһҖбһҡбһҖ бһҗбһҸ snapshot бһ“бҹғ DOM бһ…бһ»бһ… бһңбһ¶бһҷвҖӢбһўбһҖбҹ’бһҹбһҡ бһңбһ¶бһҷвҖӢбһҸбһҳбҹ’бһӣбҹғ JavaScript бһ“бһ·бһ„вҖӢбһўбһ¶бһ“ console бһ“бһ·бһ„вҖӢбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ– network бһҸбһ¶бһҳвҖӢбһҡбһҷбҹҲ socket API бһҠбһјбһ…бһӮбҹ’бһ“бһ¶бҹ” agents бһ”бҹ’бһҡбһҫвҖӢбһңбһ¶вҖӢбһҠбһҫбһҳбҹ’бһ”бһёвҖӢбһ•бҹ’бһ‘бҹҖбһ„вҖӢбһ•бҹ’бһ‘бһ¶бһҸбҹӢвҖӢбһҖбһ¶бһҡвҖӢбһ•бҹ’бһӣбһ¶бһҹбҹӢвҖӢбһ”бҹ’бһҠбһјбһҡ web бһ•бҹ’бһ‘бһ¶бһӣбҹӢвҖӢбһҒбҹ’бһӣбһҪбһ“вҖӢбһҡбһ”бһҹбҹӢвҖӢбһ–бһҪбһҖвҖӢбһңбһ¶вҖӢбһҠбҹ„бһҷвҖӢбһҳбһ·бһ“вҖӢбһ…бһ¶бҹҶвҖӢбһ”бһ¶бһ…бҹӢвҖӢбһ…бҹҒбһүвҖӢбһ–бһё cmuxбҹ” бһҹбһјбһҳвҖӢбһҳбһҫбһӣ [browser automation](https://cmux.com/docs/browser-automation)бҹ”

### бһҸбһҫ cmux бһҳбһ¶бһ“ skills бһ¬вҖӢбһ‘бҹҒ?

бһҳбһ¶бһ“бҹ” Skills бһӮбһәвҖӢбһҮбһ¶ workflows бһҠбҹӮбһӣвҖӢбһўбһ¶бһ…вҖӢбһ”бҹ’бһҡбһҫвҖӢбһЎбһҫбһ„вҖӢбһңбһ·бһүвҖӢбһҠбҹӮбһӣвҖӢбһўбҹ’бһ“бһҖвҖӢбһўбһ¶бһ…вҖӢбһ•бҹ’бһҠбһӣбҹӢвҖӢбһұбҹ’бһҷ agent бһҺбһ¶бһҳбһҪбһҷвҖӢбһҠбҹӮбһӣвҖӢбһҠбҹҶбһҺбһҫбһҡвҖӢбһҖбһ¶бһҡвҖӢбһ“бҹ…вҖӢбһҖбҹ’бһ“бһ»бһ„ cmux бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢвҖӢбһҖбһ¶бһҡбһ„бһ¶бһҡвҖӢбһҠбһјбһ…вҖӢбһҮбһ¶вҖӢбһҖбһ¶бһҡвҖӢбһ”бһүбҹ’бһҮбһ¶ CLI бһҖбһ¶бһҡвҖӢбһҹбҹ’бһңбҹҗбһҷвҖӢбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·вҖӢбһҖбһҳбҹ’бһҳ workspace бһҖбһ¶бһҡвҖӢбһҖбҹҶбһҺбһҸбҹӢ бһ“бһ·бһ„ browser surfacesбҹ” бһҡбһ»бһҖбһҡбһҖвҖӢбһҖбһ¶бһҡвҖӢбһ”бҹ’бһҡбһҳбһјбһӣвҖӢбһ•бҹ’бһҠбһ»бҹҶвҖӢбһҠбҹӮбһӣвҖӢбһ”бһҫбһҖвҖӢбһ…бҹҶбһ вҖӢбһ“бҹ… [cmux-skills](https://github.com/manaflow-ai/cmux-skills) бһ¬вҖӢбһўбһ¶бһ“ [бһҜбһҖбһҹбһ¶бһҡ skills](https://cmux.com/docs/skills)бҹ”

### бһҸбһҫвҖӢбһҒбҹ’бһүбһ»бҹҶвҖӢбһўбһ¶бһ…вҖӢбһҖбҹӮвҖӢбһҹбһҳбҹ’бһҡбһҪбһӣ keyboard shortcuts бһ”бһ¶бһ“вҖӢбһ¬вҖӢбһ‘бҹҒ?

keybindings бһҡбһ”бһҹбҹӢ terminal бһҸбҹ’бһҡбһјбһңвҖӢбһ”бһ¶бһ“вҖӢбһўбһ¶бһ“вҖӢбһ–бһёвҖӢбһҜбһҖбһҹбһ¶бһҡ config бһҡбһ”бһҹбҹӢ Ghostty (`~/.config/ghostty/config`)бҹ” shortcuts бһҮбһ¶бһҖбҹӢвҖӢбһӣбһ¶бһҖбҹӢвҖӢбһҡбһ”бһҹбҹӢ cmux (workspaces, splits, browser, notifications) бһўбһ¶бһ…вҖӢбһҖбҹӮвҖӢбһҹбһҳбҹ’бһҡбһҪбһӣвҖӢбһ”бһ¶бһ“вҖӢбһ“бҹ…вҖӢбһҖбҹ’бһ“бһ»бһ„ Settingsбҹ” бһҹбһјбһҳвҖӢбһҳбһҫбһӣ [shortcuts бһӣбҹҶбһ“бһ¶бҹҶвҖӢбһҠбһҫбһҳ](https://cmux.com/docs/keyboard-shortcuts) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢвҖӢбһ”бһүбҹ’бһҮбһёвҖӢбһ–бҹҒбһүбһӣбҹҒбһүбҹ”

### бһҸбһҫвҖӢбһҒбҹ’бһүбһ»бҹҶвҖӢбһўбһ¶бһ…вҖӢбһҖбҹӮвҖӢбһҹбһҳбҹ’бһҡбһҪбһӣ cmux бһ”бһ¶бһ“вҖӢбһ¬вҖӢбһ‘бҹҒ?

бһ”бһ¶бһ“бҹ” бһҖбһ¶бһҡвҖӢбһ”бһ„бҹ’бһ бһ¶бһү terminal бһ”бҹ’бһҡбһҫ config бһҡбһ”бһҹбҹӢ Ghostty бһҡбһ”бһҹбҹӢвҖӢбһўбҹ’бһ“бһҖ бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮ themes, fonts, бһ–бһҺбҹҢ бһ“бһ·бһ„ cursor бһҸбҹ’бһҡбһјбһңвҖӢбһ”бһ¶бһ“вҖӢбһ”бһ“бҹ’бһҸвҖӢбһҠбҹ„бһҷвҖӢбһ•бҹ’бһ‘бһ¶бһӣбҹӢбҹ” бһҖбһ¶бһҡвҖӢбһҖбҹҶбһҺбһҸбҹӢвҖӢбһ•бҹ’бһ‘бһ¶бһӣбҹӢвҖӢбһҒбҹ’бһӣбһҪбһ“вҖӢбһҡбһ”бһҹбҹӢ cmux бһ“бҹ…вҖӢбһҖбҹ’бһ“бһ»бһ„ `~/.config/cmux/cmux.json` бһӮбҹ’бһҡбһ”бҹӢвҖӢбһӮбҹ’бһҡбһ„ sidebar, tab bar, split panes бһ“бһ·бһ„вҖӢбһҘбһҡбһ·бһҷбһ¶бһ”бһҗ бһ бһҫбһҷвҖӢбһҡбһ¶бһӣбҹӢ [keyboard shortcut](https://cmux.com/docs/keyboard-shortcuts) бһўбһ¶бһ…вҖӢбһҖбҹӮвҖӢбһҹбһҳбҹ’бһҡбһҪбһӣвҖӢбһ”бһ¶бһ“бҹ” бһҹбһјбһҳвҖӢбһҳбһҫбһӣ [configuration](https://cmux.com/docs/configuration)бҹ”

### бһҸбһҫвҖӢбһҹбһҳбҹҗбһҷвҖӢбһҡбһ”бһҹбҹӢвҖӢбһҒбҹ’бһүбһ»бҹҶвҖӢбһҸбҹ’бһҡбһјбһңвҖӢбһ”бһ¶бһ“вҖӢбһҡбһҖбҹ’бһҹбһ¶вҖӢбһ‘бһ»бһҖвҖӢбһ¬вҖӢбһ‘бҹҒ?

бһҡбһҖбҹ’бһҹбһ¶вҖӢбһ‘бһ»бһҖбҹ” cmux бһҹбҹ’бһҠбһ¶бһҡ бһ”бһ„бҹ’бһўбһҪбһ… workspaces panes бһҗбһҸвҖӢбһҖбһ¶бһҡбһ„бһ¶бһҡ бһ“бһ·бһ„ scrollback бһҡбһ”бһҹбҹӢвҖӢбһўбҹ’бһ“бһҖвҖӢбһ“бҹ…вҖӢбһ–бҹҒбһӣвҖӢбһўбҹ’бһ“бһҖвҖӢбһ”бһҫбһҖвҖӢбһЎбһҫбһ„вҖӢбһңбһ·бһү бһ бһҫбһҷвҖӢбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–вҖӢбһ“бҹ…вҖӢбһҡбһҹбҹӢвҖӢбһҡбһ¶бһ“вҖӢбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢвҖӢбһ–бһёвҖӢбһҖбһ¶бһҡвҖӢбһ”бһҫбһҖвҖӢбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡвҖӢбһЎбһҫбһ„вҖӢбһңбһ·бһүвҖӢбһ–бҹҒбһүбһӣбҹҒбһү бһҳбһ·бһ“вҖӢбһҳбҹӮбһ“вҖӢбһӮбҹ’бһҡбһ¶бһ“бҹӢвҖӢбһҸбҹӮвҖӢбһҖбһ¶бһҡвҖӢбһ”бһ·бһ‘вҖӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёвҖӢбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮвҖӢбһ‘бҹҒбҹ” бһҹбһҳбҹҗбһҷ agent бһҠбһјбһ…вҖӢбһҮбһ¶ Claude Code, Codex бһ“бһ·бһ„ OpenCode бһҸбҹ’бһҡбһЎбһ”бҹӢвҖӢбһҳбһҖвҖӢбһңбһ·бһүвҖӢбһ•бһ„вҖӢбһҠбҹӮбһҡбҹ” бһҹбһјбһҳвҖӢбһҳбһҫбһӣ [session restore](https://cmux.com/docs/session-restore)бҹ”

### бһҸбһҫвҖӢбһңбһ¶вҖӢбһ”бҹ’бһҡбҹҖбһ”вҖӢбһ’бҹҖбһ”вҖӢбһ“бһ№бһ„ tmux бһҷбҹүбһ¶бһ„вҖӢбһҠбһјбһ…бһҳбҹ’бһҠбҹҒбһ…?

tmux бһӮбһәвҖӢбһҮбһ¶ terminal multiplexer бһҠбҹӮбһӣвҖӢбһҠбҹҶбһҺбһҫбһҡвҖӢбһҖбһ¶бһҡвҖӢбһ“бҹ…вҖӢбһҖбҹ’бһ“бһ»бһ„ terminal бһҺбһ¶бһҳбһҪбһҷбҹ” cmux бһӮбһәвҖӢбһҮбһ¶вҖӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё native macOS бһҠбҹӮбһӣвҖӢбһҳбһ¶бһ“ GUIбҹ– vertical tabs, split panes, browser бһҠбҹӮбһӣвҖӢбһ”бһ„бҹ’бһҖбһ”бҹӢ бһ“бһ·бһ„ socket API бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢвҖӢбһҳбһ¶бһ“вҖӢбһҹбҹ’бһҡбһ¶бһ”бҹӢ бһҠбҹ„бһҷвҖӢбһҳбһ·бһ“вҖӢбһ…бһ¶бҹҶвҖӢбһ”бһ¶бһ…бҹӢвҖӢбһҳбһ¶бһ“вҖӢбһҜбһҖбһҹбһ¶бһҡ config бһ¬ prefix keysбҹ” бһ‘бҹ„бҹҮвҖӢбһҮбһ¶вҖӢбһҷбҹүбһ¶бһ„вҖӢбһҺбһ¶вҖӢбһҖбҹҸвҖӢбһҠбҹ„бһҷ бһҳбһ“бһ»бһҹбҹ’бһҹвҖӢбһҮбһ¶вҖӢбһ…бҹ’бһҡбһҫбһ“вҖӢбһҠбҹҶбһҺбһҫбһҡвҖӢбһҖбһ¶бһҡ cmux бһҮбһ¶бһҳбһҪбһҷ SSH бһ“бһ·бһ„ tmux бһҡбһҪбһҳвҖӢбһӮбҹ’бһ“бһ¶вҖӢбһҠбҹ„бһҷвҖӢбһҹбһ”бҹ’бһ”бһ¶бһҷвҖӢбһҡбһёбһҖбһҡбһ¶бһҷ бһ бһҫбһҷ cmux бһўбһ¶бһ…вҖӢбһ—бҹ’бһҮбһ¶бһ”бҹӢвҖӢбһ‘бҹ…вҖӢбһҹбһҳбҹҗбһҷ tmux бһ–бһёвҖӢбһ…бһҳбҹ’бһ„бһ¶бһҷвҖӢбһҡбһ”бһҹбҹӢвҖӢбһўбҹ’бһ“бһҖвҖӢбһҠбҹ„бһҷ native ([beta](https://cmux.com/docs/remote-tmux))бҹ”

### бһҸбһҫ cmux бһҘбһҸвҖӢбһӮбһ·бһҸвҖӢбһҗбҹ’бһӣбҹғвҖӢбһ¬вҖӢбһ‘бҹҒ?

бһҳбҹӮбһ“ cmux бһҘбһҸвҖӢбһӮбһ·бһҸвҖӢбһҗбҹ’бһӣбҹғвҖӢбһҖбҹ’бһ“бһ»бһ„вҖӢбһҖбһ¶бһҡвҖӢбһ”бҹ’бһҡбһҫвҖӢбһ”бҹ’бһҡбһ¶бһҹбҹӢбҹ” бһҖбһјбһҠвҖӢбһ”бҹ’бһҡбһ—бһ–вҖӢбһҳбһ¶бһ“вҖӢбһ“бҹ…вҖӢбһӣбһҫ [GitHub](https://github.com/manaflow-ai/cmux)бҹ”

### бһҸбһҫвҖӢбһҒбҹ’бһүбһ»бҹҶвҖӢбһўбһ¶бһ…вҖӢбһӮбһ¶бҹҶбһ‘бҹ’бһҡ cmux бһҷбҹүбһ¶бһ„вҖӢбһҠбһјбһ…бһҳбҹ’бһҠбҹҒбһ…?
